Master Zorin OS

The ultimate future-proof guide to installing, configuring, and optimizing Zorin OS for any PC or laptop.

Welcome to the complete guide for setting up Zorin OS. Whether you're moving away from Windows or upgrading, this guide covers everything from writing the ISO to applying post-install tweaks for maximum performance.

1. How to Install Zorin OS

The installation process is highly user-friendly. Follow these steps to ensure a clean, reliable installation on any system.

1 Download & Flash

Go to the Zorin OS Download page. Download the Core (free) version. Flash it to a USB (8GB+) using balenaEtcher or Rufus.

2 Boot & Install

Restart and tap your Boot Menu key (F12, F8, F2). Select the USB. Choose "Try or Install Zorin OS". Once loaded, click the "Install" icon, connect to Wi-Fi, and check "Install third-party software for graphics".

2. Essential Post-Install Setup

1. Update the System

sudo apt update && sudo apt upgrade -y

2. Essential Drivers

Open the menu, search Additional Drivers. Select proprietary NVIDIA/AMD drivers and click Apply.

3. Enable Firewall

sudo ufw enable

4. Media Codecs

sudo apt install ubuntu-restricted-extras -y

3. Install Archive Managers

Install core backends for encrypted archives, and a powerful GUI.

1. Backend & PeaZip GUI

sudo apt install p7zip-full p7zip-rar -y
flatpak install flathub io.github.peazip.PeaZip -y

4. What Veterans Wish They Knew

The "Ubuntu Base" Secret

Zorin OS is built on Ubuntu LTS. If you have an issue, search for "Ubuntu [issue]" instead of Zorin. The solutions will work perfectly.

Flatpaks > PPAs

Avoid adding custom "PPAs" for software. Always prefer installing apps via Flatpak from the Software store to keep your system stable.

5. Performance Tweaks

Important Disclaimer

These tweaks are optional and meant for specific hardware scenarios. Read carefully before applying.

Best for older HDDs

1. Install Preload

Caches apps in unused RAM. Do not use on fast NVMe SSDs.

sudo apt install preload -y
Best for Low RAM (≤4GB)

2. Enable zRAM

Compresses data in RAM instead of writing to slow disk swap.

sudo apt install zram-tools -y

Master Linux Mint

The comprehensive guide to installing, configuring, and unleashing the power of Linux Mint Cinnamon.

Welcome to the complete guide for Linux Mint. Renowned for its rock-solid stability, refusal to force Snaps on users, and the highly customizable Cinnamon desktop environment.

1. How to Install Linux Mint

1 Download Cinnamon Edition

Go to the Linux Mint Download page. Select the "Cinnamon Edition" as it provides the most modern, complete experience. Flash it to a USB drive using balenaEtcher.

2 Boot & Install

Boot from your USB. Once the Mint desktop loads, double-click "Install Linux Mint". Crucially, when prompted, tick the box that says "Install multimedia codecs" to ensure videos and music play perfectly out of the box.

2. Essential Post-Install Setup

Linux Mint has exceptional GUI tools. While you can use the terminal, Mint encourages using its built-in managers.

1. Update Manager

Look for the shield icon in your bottom-right system tray. Click it, apply the update to the manager itself if prompted, and then install all available system updates.

2. Timeshift (Backups)

Unlike other distros, Mint comes with Timeshift pre-installed. Open it from the menu, select RSYNC, and set up daily automated snapshots. Do this before changing drivers!

3. Driver Manager

Open the Mint Menu and search for "Driver Manager". It will automatically find and let you install the required NVIDIA proprietary drivers or Wi-Fi firmware.

4. Redshift (Night Light)

Mint uses an applet called "Redshift" for blue-light filtering. Search for it in the menu, open it, and it will sit in your tray, protecting your eyes at night.

5. Backup Codecs Command

If you forgot to tick the codecs box during installation, you must run this custom Mint command:

sudo apt install mint-meta-codecs -y

3. Archive Tools

Mint's default archive manager is quite good, but for deep compatibility, add the 7zip backends and PeaZip.

sudo apt install p7zip-full p7zip-rar -y
flatpak install flathub io.github.peazip.PeaZip -y

4. What Veterans Wish They Knew

The Anti-Snap Philosophy

Unlike Ubuntu, Linux Mint actively blocks the installation of "Snaps" (a type of software package) because they are proprietary to Canonical. Mint fully embraces Flatpak instead. Always use the "Software Manager" to find Flatpak versions of your apps.

Cinnamon Spices

You don't need browser extensions to customize Mint. Open settings and look for Applets (taskbar items), Desklets (desktop widgets), and Extensions. You can download hundreds of community-made tools directly from within these settings panels.

Nemo File Manager Superpowers

Mint uses "Nemo" as its file manager. Open Nemo, go to Edit > Preferences > Behavior, and enable "Open in Terminal". Also, check out Nemo Actions in your software manager to add root-access options and advanced right-click context menus directly to your file browser.

5. Performance Tweaks

Important Disclaimer

Linux Mint is extremely light and stable by default. Only apply these if you are on very old hardware or specifically need battery improvements.

Best for Laptops (Battery/Heat)

1. Auto-CPUFreq

Mint uses older power profiles sometimes. This tool perfectly manages fan speed and CPU throttling.

sudo snap install auto-cpufreq

Wait! Mint blocks snaps. To use this specific tool on Mint, you must unblock snaps first: sudo rm /etc/apt/preferences.d/nosnap.pref then sudo apt install snapd.

Best for Low RAM (≤4GB)

2. Enable zRAM

Compresses data in RAM. Crucial for keeping Mint snappy on old dual-core laptops.

sudo apt install zram-tools -y
